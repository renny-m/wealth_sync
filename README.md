# Wealth Sync

Wealth Syncは、個人の資産を簡単に管理・追跡できる資産管理ツールです。目標額に対する実績の可視化や、月ごとの資産状況を一目で確認できる機能を提供します。

## 主な機能
- **メイン画面**: 
  - 当月の資産総額目標と実績の表示
  - 資産配分目標と実績を表示し、リバランスの必要性を表示
- **目標管理画面**: 
  - 運用期間、支出金額、資産配分目標、月ごとの積立目標の入力機能
- **資産実績管理画面**: 
  - 各資産の月ごとの目標金額を表示し、実績の入力をサポート

## 技術スタック
- フロントエンド: TypeScript, React, Next.js
- バックエンド: Ruby, Ruby on Rails, GraphQL
- データベース & 認証: Firebase
